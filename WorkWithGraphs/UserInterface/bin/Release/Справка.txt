Для работа с данной программой необходимо указать размер матрицы. Этот размер Вы можете выбрать на главном окне программы.

После выбора размера нужно заполнить таблицу смежности будущего графа, по которой программа построит модель этого графа.

Заполнять поля нужно исключительно 0 и 1.

Чтобы иметь возможность работы с программой, необходимо ввести оба графа и нажать у каждого на кнопку "Создать граф".

После создания Вам будут доступны любые операции над графами.

В первом пункте меню "Операции над графами" можно получить информацию по каждому графу из соответствующего выбраного подпункта меню.

Во втором пункте меню "Операции над подграфами" можно получить дданные об изолированных или дополняющих подграфах для данных графов. Эти данные нужно будет сохранить на диске в виде тестовго файла. Для этого программа попросит Вас указать путь, куда будет сохранен файл с полученными данными. Данная операция выполняется для вывода результатов:
Пересечения;
Объединения;
Поиска всех правильных графов;
Поиск дополняющего подграфа до полного.

Все числовые данные выводятся на окне программы сверху от таблицы смежности соответствующего графа.
