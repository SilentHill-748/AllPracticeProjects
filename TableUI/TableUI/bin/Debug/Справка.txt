Для работы с данным приложением необходимо нажать ЛКМ на "Операции над таблицей" и в открывшемся окне в соответствующем поле начать вводить данные.

В данном окне есть 5 видов операций: поиск, добавление, изменение, удаление и выборка данных из таблицы.

	Для добавления записи нужно заполнить поля "Номер страницы" и "Номер строки", а поле "Текст для изменения строки" может быть пустым. Для добавления нужно нажать кнопку "Добавить".
	Для редактирования аналогчно, но нужно обязательно указать номер записи. Введенные данные перезапишут те данные, которые были в той записи, которую планируете заменить.
	Удаление записей производится по их номеру из выпадающего списка "Номер записи для удаления". Для удаления надо выбрать нужный номер и нажать на кнопку "Удалить".
	Выборка осуществляется путем заполнения 3 полей. Первым (слева направо) полем является название столбца. Второе является условием отбора (>, <, >=, <=, =, !=). Третье - значение столбца. После ввода всех данных нужно нажать кнопку "Выбрать".
	Программа поддерживает 3 вида поиска: по значению столбца, минимальное значение столбца и поиск пустых значений столбцов в записи. Чтобы найти запись по значению её столбца, нужно указать само значение и название столбца. Для поиска минимального значения в столбце - указать название столбца (значение не нужно). Для поиска пустых значение ничего вводить не надо. После выбора, какой вид поиска хотите использовать и ввода данных - нажмите на соответствующую кнопку: "по значению" - поиск запаиси по введенному значению в указанном столбце; "Мин. значение в столбце" - поиск записи с наименьшим значением в указанном столбце; "Записи с пустыми значениями" - начнет поиск всех записей, где хотя бы 1 значение любого столбца пустое.

	Также программа поддерживает сортировку данных. Для этого в главном окне выберите меню "Сортировка" и в нем выберите нужное условие сортировки.

	Если нужно удалить все записи в таблице - выберите на главной форме меню "Очистка данных" и "Очистка всех данных".
	
	Меню "Справка" открывает данный файл.
